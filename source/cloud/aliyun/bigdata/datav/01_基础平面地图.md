### 第三步 对基础平面地图进行数据配置——显示上海市

为了让地图显示上海，我们需要获取上海市的坐标位置。可以通过[**高德地图开发者工具——坐标拾取器**](https://lbs.amap.com/api/) 来获取。

登陆https://lbs.amap.com/api/ 选择 "坐标拾取器"

![](pic/006.png)

![](pic/007.png)

坐标拾取器：详细地址与精准坐标之间的相互转化。如下图，输入上海，右侧坐标获取结果即可显示对应的坐标信息。

![](pic/008.png)



```shell
[
  {
    "zoom" : 10,
    "lng": 121.473658,
    "lat": 31.230378
  }
]
```
